# Personal AI OS

Telegram-native multi-agent personal assistant with Claude API.

## Stack
- Python 3.12, FastAPI, python-telegram-bot
- Anthropic Claude (sonnet + haiku with smart routing)
- PostgreSQL (Supabase), Redis, Celery
- Docker Compose, nginx + Let's Encrypt

## Quick start
See [LAUNCH.md](LAUNCH.md) for full step-by-step deployment guide.

Private product docs (BRD, context handoff, credentials) live in **`docs-private/`** on your machine only — that folder is gitignored.
